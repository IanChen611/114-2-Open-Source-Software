# Userspace 驗證器

## 編譯

```bash
make userspace
```

## 執行（使用內建棋盤）

```bash
./sudoku_validator
```

## 或傳入自訂棋盤檔案

```bash
./sudoku_validator test-board/correct-board.txt
```

# Kernel Module（sudoku.ko）

## 前置作業：取得 WSL2 Kernel 原始碼並編譯（僅需做一次）

```bash
# 下載對應版本的 WSL2 kernel source
git clone --depth=1 --branch linux-msft-wsl-6.6.114.1 \
    https://github.com/microsoft/WSL2-Linux-Kernel.git ~/WSL2-Linux-Kernel

# 安裝編譯依賴
sudo apt install build-essential flex bison libssl-dev libelf-dev bc dwarves

# 套用目前執行中的 kernel config
cd ~/WSL2-Linux-Kernel
cp /proc/config.gz . && gunzip config.gz && mv config .config
make olddefconfig

# 完整編譯（約 30–60 分鐘，產生 Module.symvers）
make -j$(nproc)
```

## 編譯 Kernel Module

```bash
# 回到專案目錄
cd /path/to/114-2-Open-Source-Software

make
```

> 若出現 `Module.symvers is missing` 警告但 kernel source 尚未完整編譯完成，
> 可暫時使用以下指令跳過 symbol 版本驗證：
> ```bash
> KBUILD_MODPOST_WARN=1 make
> ```

## 載入 Kernel Module

```bash
sudo insmod sudoku_module.ko

# 確認 /dev/sudoku 已出現
ls /dev/sudoku

# 查看 kernel log 確認載入成功
dmesg | tail -5
```

## 啟動 CLI 遊戲

```bash
# 編譯 CLI 客戶端（只需做一次）
make userspace

# 啟動遊戲
sudo ./sudoku-cli
```

### 遊戲指令

| 指令 | 說明 |
|---|---|
| `列 欄 數字` | 填入數字（1–9 為 1-indexed；0 代表清除） |
| `hint` | 自動填入一格正確答案 |
| `reset` | 還原棋盤至初始謎題 |
| `new` | 開始新遊戲 |
| `quit` | 離開程式 |

## 卸載 Kernel Module

```bash
sudo rmmod sudoku_module

# 確認已卸載
dmesg | tail -3
```

## 清除編譯產物

```bash
make clean
```
